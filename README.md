# CyberSec

A collection of cybersecurity tools and simulations built for learning, interview prep, and role-based training.

---

## Projects

### SOC-Simulator — Stratum SecOps

An interactive, browser-based Security Operations Center (SOC) platform simulator. No build step required — open one HTML file and you're running a full simulated SOC environment with realistic role-based dashboards, live incident data, and an altitude view that shows how the same incident looks at every level of the organization.

[View simulator details →](./SOC-Simulator/README.md)

**Quick start:** Open `SOC-Simulator/Stratum SecOps.html` in any modern browser.

![Manager Home Dashboard](./SOC-Simulator/screenshots/01-manager-home.png)

### Topicgen

## Purpose

This repo exists to build and sharpen practical cybersecurity skills through simulation, tooling, and structured practice. Each project targets a specific gap — whether that's understanding how a SOC manager thinks, how detections are engineered, or how incidents are communicated up the executive chain.
